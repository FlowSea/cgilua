Cookies

cookies.get (name)

Gets the value of the named cookie

cookies.set (name, value[, options])

Sets the value of the named cookie. This function should be called before the HTTP header is sent.

cookies.sethtml (name, value[, options])

Sets the value of the named cookie. This function generates a <meta> HTML element so it should be called after the <head> HTML tag was produced and before the <body>.

cookies.delete (name[, options])

Deletes the value of the named cookie by setting its value to xxx.

File System

filesystem.chdir (path)

Changes the current working directory.

filesystem.currentdir ()

Returns the current working directory or nil plus an error string.

filesystem.dir (path)

Lua iterator over the entries of a given directory.

filesystem.lock (filehandle, mode[, start[, length]])

Locks a file or a part of it. This function works on open files; the file handle should be specified as the first argument. The string mode could be either r (for a read/shared lock) or w (for a write/exclusive lock). The optional arguments start and length can be used to specify a starting point and its length; both should be numbers. This function returns a boolean indicating if the operation was successful; in case of error, it returns false plus a string describing the error.

filesystem.unlock (filehandle[, start[, length]])

Unlocks a file or a part of it. This function works on open files; the file handle should be specified as the first argument. The optional arguments start and length can be used to specify a starting point and its length; both should be numbers. This function returns a boolean indicating if the operation was successful; in case of error, it returns false plus a string describing the error.

Pre-processing

translate (string)

Creates a string with Lua code that reproduces the given string executing the Lua chunks or expressions inside it.

setcompatmode (boolean)

Turns on or off the compatibility mode. Turning it on will make the translator understands the old expression fields and code fields structures, now obsolete.

setoutfunc (string)

Defines the output function. The translator will write calls to the function with the given name.

URL-encode

url_unescape (str)

Decode an URL-encoded string following RFC-2396.

url_escape (str)

URL-encode a string following RFC-2396.

url_insertfield (args, name, value)

Insert a (name=value) pair into table args.

url_parsequery (query, args)

Parse URL-encoded request data (the query part of the script URL or URL-encoded POST data). Each decoded (name=value) pair is inserted into table args.

url_encodetable (args)

URL-encode the elements of a table creating a string to be used in an URL for passing data/parameters to another script.
