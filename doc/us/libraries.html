Distributed libraries

cookies · directories · HTML pre-processing · POST data processing · URL encoding

Cookies

cookie.get (name)

Gets the value of the named cookie

cookie.set (name, value, options)

Sets the value of the named cookie. This function should be called before the HTTP header was sent.

cookie.sethtml (name, value, options)

Sets the value of the named cookie. This function generates a <meta> HTML element so it should be called after the <head> HTML tag was produced and before the <body>.

cookie.delete (name, options)

Deletes the value of the named cookie by setting its value to xxx.

Directories

dir.chdir (path)

Changes the current working directory.

dir.getcwd ()

Returns the current working directory or nil plus an error string.

Pre-processing

translate (string)

Create a string with Lua code that reproduce the given string executing the Lua chunks or expressions inside it.

setcompatmode (boolean)

Turns on or off the compatibility mode. Turning it on will make the translator understands the old expression fields and code fields structures, now obsolete.

setoutfunc (string)

Defines the output function. The translator will write calls to the function with the given name.

URL-encode

url_unescape (str)

Decode an URL-encoded string following RFC-2396.

url_escape (str)

URL-encode a string following RFC-2396.

url_insertfield (args, name, value)

Insert a (name=value) pair into table args.

url_parsequery (query, args)

Parse URL-encoded request data (the query part of the script URL or URL-encoded POST data). Each decoded (name=value) pair is inserted into table args.

url_encodetable (args)

URL-encode the elements of a table creating a string to be used in an URL for passing data/parameters to another script.
